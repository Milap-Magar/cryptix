## 🚀 Overview

Cryptix is a modern landing page inspired by Framer templates, featuring sections including:

* Hero Section
* Features
* What We Do
* Story
* FAQ
* Pricing
* Footer

## 🎯 Why I Made This Project

The primary purpose of building Cryptix was to:

* Enhance my CSS and responsive design skills.
* Gain hands-on experience in modern UI/UX development.
* Understand component-based architecture and landing page design patterns.
* Improve my ability to recreate and customize professional website designs.
